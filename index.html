Protocolo de Canje

1

Inicia el Proceso

Ubica tu PIN y prepárate para ingresarlo en la terminal de acceso.

2

Datos de Usuario

Ingresa tus datos personales. Debes ser mayor de 18 años o contar con autorización.

3

Verificación de ID

Ingresa tu ID de jugador. IMPORTANTE: Haz clic en "Verificar ID" antes de canjear. Asegúrate de que el nombre coincida.

⚠
ADVERTENCIA:

Si procedes con un ID incorrecto, no podremos revertir el canje. Verifica cuidadosamente. Se requiere evidencia para soporte.
